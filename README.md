# **📋 Описание проекта**
Codebattle — это Open Source платформа для проведения соревнований по программированию в реальном времени. Проект создан сообществом Hexlet и позволяет разработчикам соревноваться друг с другом в скорости и правильности решения алгоритмических задач.

# **🚀 Этапы работы над проектом**
##В рамках данного репозитория реализован полный цикл тестирования (STLC):

### 1. Анализ и классификация требований
На начальном этапе была проведена декомпозиция проекта. Требования заказчика были проанализированы и разделены на функциональные и нефункциональные. Это позволило определить границы тестирования (Scope) и приоритетные области.

### 2. Разработка тестовой документации
## На основе уточненных требований был спроектирован набор тест-кейсов.

* Покрытие позитивных и негативных сценариев.

* Проверка критического пути пользователя (создание игры, участие в битве).

* Кроссбраузерное тестирование интерфейса.

### 3. Выполнение тестирования и Bug Reporting
## В ходе активной фазы тестирования были выявлены дефекты различного уровня критичности.

*Для каждого бага составлен подробный баг-репорт.

*Логика описания: environment -> Steps to reproduce -> Expected result -> Actual result.

*Приложены скриншоты и логи (при необходимости).

### 4. Регрессионное тестирование
После внесения исправлений (или имитации процесса исправления) был проведен финальный этап регрессионного тестирования. Целью этапа была проверка того, что новые изменения не повлияли на работоспособность существующего функционала.
